## Features

- **User Authentication**: Secure login and registration functionality.
- **Responsive UI**: Adaptable to both Android and iOS devices.
- **State Management**: Efficient use of provider for state management.
- **Network Requests**: Asynchronous data fetching with error handling.
- **Local Storage**: Persistent data storage using shared_preferences.
- **Theming**: Customizable themes for a personalized user experience.
